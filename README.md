# 기능 목록

### 공통 확인 부분
1. 자바 코드 컨벤션 확인
2. indent(인덴트, 들여쓰기) depth 3이 넘지 않도록 확인
3. 3항 연산자 확인(사용불가)
4. 함수(또는 메소드)가 한가지 일만 하도록 최대한 작게 만들기

### 기능 구현 순서
1. 정답 값 생성
2. 사용자에게 임의의 값 입력 받기
3. 스트라이크 개수 세기
4. 볼 개수 세기
5. 스크라이크와 볼이 없을 경우 낫싱 처리
6. 결과 출력
7. 정답인 경우 정답메세지 출력
8. 정답인 경우 사용자에게 게임 진행여부 입력받기
9. 게임 다시시작 혹은 게임 종료
